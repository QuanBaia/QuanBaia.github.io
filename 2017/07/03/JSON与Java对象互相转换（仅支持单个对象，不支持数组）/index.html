JSON与Java对象互相转换

今天突然想起以前在做项目的时候总是会遇到关于Json转换的问题，然后就写一个关于Json转换的一个工具类，这样我们遇到Json转换的问题就可以很方便的进行转换

优点

  • 使用static静态修饰，在其他类（方法）中直接调用避免创建重复对象
  • 对象与Json之间直接转换

缺点

  • 本方法只试用于单个对象，进行数组（List集合）转换时会报错

  • ezmorph-1.0.4.jar
  • jdom.jar
  • json-lib-2.2.2-jdk15.jar
  • json-lib-2.2.3-jdk13.jar
